The error message you 're seeing is due to the fact that you' re redeclaring the template parameter J in the visit_grouping_expression method.This is not allowed in C++ as it causes a shadowing issue.

    To fix this,
    you should remove the template declaration from the visit_grouping_expression method.The corrected code should look like this :

    Please make sure to replace J with the appropriate type when you call
    visit_grouping_expression()
        .

    lass grouping_expression : public expression {
   public:
    grouping_expression(std::unique_ptr<expression> expression);
    template <typename J>
    J accept();
    J visit_grouping_expression();

   private:
    std::unique_ptr<expression> _expression;
};
